# Navinta AI — Cleanup Report
**Date:** 2026-02-21

## Project Overview
- **Framework:** Vite + React 18 (client), Express.js + TypeScript (server), Expo SDK 52 (mobile)
- **Entry Points:** `client/src/main.tsx`, `server/index.ts`, `mobile/app/`
- **Build:** `vite build` (client), `tsx server/index.ts` (server)
- **Dev:** `npm run legacy:dev` → concurrent Vite + Express

### `legacy/` — Old project copy (276 files, 416MB node_modules)
Entire previous version of the app before monorepo restructuring. Contains its own client, server, shared, scripts, and a full node_modules. **Zero references** from active code.

### `apps/web/` — Abandoned Next.js rewrite (51 files, 12MB node_modules)
A Next.js version of the web app (has `next.config.ts`, `/app` directory routing, NextAuth). Never connected to the active Vite+Express codebase. **Zero references** from active code.

### `apps/mobile/` — Duplicate Expo app (143MB node_modules)
Turborepo workspace copy of the mobile app. The active mobile app lives at `mobile/`. **Zero references** from active code.

### `packages/` — Unused turborepo packages (17 files)
Shared `tsconfig` and `ui` packages from turborepo setup. **Zero references** from active code — the app uses its own `client/src/components/ui/` and root `tsconfig.json`.

### `dist/` — Build output (4 files, 1.4MB)
Regenerated by `vite build`. Safe to delete.

### `turbo.json` — Turborepo config
No longer needed since workspaces and turbo commands were removed.

### `.turbo/` — Turborepo cache
Build cache for the now-removed turborepo setup.
