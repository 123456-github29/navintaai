DATA MODEL

type Word = {
  id: string
  text: string
  start: number // seconds
  end: number   // seconds
  confidence?: number
}

type CaptionSegment = {
  id: string
  start: number
  end: number
  words: Word[]
}

Implement:
- A parser to convert Whisper-style output to Word[]
- A segmenter that groups words into readable segments (max words per line, max duration, punctuation-aware, avoid single-word lines)
- A timeline engine that, given currentTime, returns active segment + active word index + progress within word
