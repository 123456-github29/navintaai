D) JOB WORKER FLOW (THE CORRECT ORDER)
========================
When a render job starts:

D1) Verify user owns the video + clip path(s)
D2) Create signed URL for the raw compiled video OR the clip you want to auto-cut
   - TTL 2–5 minutes
D3) Call Python to generate EDL
D4) Feed EDL into Remotion rendering
D5) Upload MP4 to Supabase renders bucket
D6) Mark job finished and return signed URL for output MP4 (TTL 10–30 minutes)

IMPORTANT:
- The Python service is ONLY for EDL creation.
- Remotion is ONLY for rendering.
- Supabase is ONLY for storage + auth.
