================================================================================

Create src/agents/cinematographerAgent.ts that produces EditDecisionJSON from real clips.

Input:
- clips[]: each has storage_path, duration_seconds, shot metadata
- post: includes title, shotList, musicVibe, etc.
- userStyle: aspect ratio target and style selections

  3) Adds smooth transitions:
     - crossfade or blur-dissolve 8–12 frames
  4) Adds basic color preset per plan:
     - free/starter: "clean"
     - pro/studio: allow "vivid" or "cinematic"
  5) Captions:
     - Use existing transcription pipeline OR placeholder for now:
       - If transcript exists: generate caption chunks 2–6 words each
       - If not: still allow rendering without captions
     - Use karaoke highlight style for pro/studio; simple pop-in for free/starter

Important:
- Validate output against EditDecisionJSONSchema
- Return deterministic, consistent results (no randomness unless seeded)

Asset Map:
- Build assetMap { assetId: signedUrl } for each clip
- assetId should be stable: clip.id
