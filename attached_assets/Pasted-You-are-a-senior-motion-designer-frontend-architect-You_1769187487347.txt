GLOBAL ANIMATION RULES
========================
1. Use GSAP ONLY. Remove or disable any Tailwind/CSS/Framer transitions that conflict.

3. Use these global timing tokens:
   - SLOW = 1.0s

5. Add breathing room (delays) so animations don’t feel rushed.

========================
ANIMATION HIERARCHY
========================
Every section MUST follow this order:
1. Background / container (SLOW)
2. Headline (MEDIUM)
3. Subtext (MEDIUM, delayed)
4. Visuals (MEDIUM, delayed more)
5. CTA buttons (FAST, last)

NO exceptions.

========================
SECTION LOAD BEHAVIOR
========================

- Create ONE GSAP timeline (timeline per section)

- Add 0.2–0.4s pause between groups
- Never animate everything at once
- Text always comes BEFORE visuals
- CTA always comes LAST

========================
SCROLL ANIMATIONS
========================

- Remove any autoplay scroll animations

========================
PAGE TRANSITIONS
========================

========================
MICRO INTERACTIONS
